Popis:
Mějme libovolný HTML dokument. Vytvořte skript, který při vložení do HTML dokumentu zobrazí vedle obsahu stránky informační panel zobrazující elementy dokumentu ve formě DOM stromu. Kliknutím na libovolný uzel DOM stromu se vizuálně zvýrazní příslušný element ve zobrazené stránce a opačně - kliknutím na libovolné místo stránky se vybere příslušný uzel ve stromu. Pro vybraný uzel stromu umožněte editaci jeho atributů class a id. Implementujte funkci getSelectedElement(), která vrací právě vybraný DOM element nebo null, pokud nic není vybráno. Uvažujte pouze uzly DOM stromu typu Document, Element a Text. Uzly ostatních typů je možno ignorovat.

Nejaky iny text, do ktoreho vlozim dalsi element zvyraznenia, aby som mal nejake tie zanorene struktury.
